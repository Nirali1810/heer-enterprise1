TECHNICAL SPECIFICATIONS
=================================================

TECHNOLOGY STACK:
- Frontend: React/Vue.js
- Backend: Node.js + Express.js
- Database: MongoDB/SQL
- Payment: Razorpay API v1
- Security: HMAC-SHA256, JWT, bcrypt
- Email: Nodemailer
- HTTP: REST API

SUPPORTED PAYMENT METHODS:
✅ Credit Cards (Visa, Mastercard)
✅ Debit Cards
✅ UPI (India)
✅ NetBanking (India)
✅ Digital Wallets
✅ EMI (Select banks)

API ENDPOINTS:
POST   /api/payment/create-order       → Create order
POST   /api/payment/verify              → Verify payment
GET    /api/payment/:paymentId         → Get payment details
GET    /api/order/:orderId             → Get order details
POST   /api/payment/refund             → Process refund
POST   /api/payment/webhook (optional) → Handle webhooks

HTTP STATUS CODES:
200 → Success
201 → Created
400 → Bad Request (validation error)
401 → Unauthorized (auth failure)
403 → Forbidden (CORS/access)
404 → Not Found (order/payment)
429 → Rate Limited
500 → Server Error
